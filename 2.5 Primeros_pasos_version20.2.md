## ¬øQu√© hay de nuevo en Angular 20.2?

Angular 20.2 introduce cambios significativos en la estructura del proyecto, simplificando a√∫n m√°s el desarrollo:

- **Nueva estructura simplificada**: Ya no se usa `app.component`, ahora es simplemente `app`
- **Componentes standalone por defecto**: Todos los componentes son standalone autom√°ticamente
- **Bootstrap simplificado**: El proceso de arranque de la aplicaci√≥n es m√°s directo
- **Mejor tree-shaking**: Optimizaci√≥n autom√°tica mejorada
- **TypeScript 5.7+**: Soporte completo para las √∫ltimas caracter√≠sticas de TypeScript

## 6. Trabajando con Componentes en Angular 20.2

### ¬øQu√© son los componentes?

Los componentes son las **piezas fundamentales** de una aplicaci√≥n Angular. Cada componente encapsula:

- **Template (HTML)**: La estructura visual
- **Estilos (CSS)**: La apariencia
- **L√≥gica (TypeScript)**: El comportamiento y datos
- **Metadatos**: Configuraci√≥n del componente

**Piensa en los componentes como bloques de LEGO reutilizables** que puedes combinar para construir tu aplicaci√≥n.

### ¬øPara qu√© sirven los componentes?

1. **Reutilizaci√≥n**: Crear una vez, usar en muchos lugares
2. **Organizaci√≥n**: Dividir la aplicaci√≥n en partes manejables
3. **Mantenimiento**: Cada componente tiene una responsabilidad espec√≠fica
4. **Testeo**: M√°s f√°cil probar componentes individuales
5. **Trabajo en equipo**: Diferentes desarrolladores pueden trabajar en componentes separados

#### **contador.ts**
```typescript
import { Component, signal, input, output, OnInit } from '@angular/core';
import { CommonModule } from '@angular/common';

@Component({
  selector: 'app-contador',
  standalone: true,
  imports: [CommonModule],
  templateUrl: './contador.html',
  styleUrl: './contador.css'
})
export class Contador implements OnInit {
  // Inputs - datos que recibe el componente
  valorInicial = input(0);
  titulo = input('Contador');
  maximo = input(100);
  
  // Outputs - eventos que emite el componente
  cambioValor = output<number>();
  
  // Estado interno del componente
  valor = signal(0);
  
  ngOnInit() {
    // Inicializar con el valor recibido
    this.valor.set(this.valorInicial());
  }
  
  incrementar() {
    const nuevoValor = this.valor() + 1;
    if (nuevoValor <= this.maximo()) {
      this.valor.set(nuevoValor);
      this.cambioValor.emit(nuevoValor);
    }
  }
  
  decrementar() {
    const nuevoValor = this.valor() - 1;
    if (nuevoValor >= 0) {
      this.valor.set(nuevoValor);
      this.cambioValor.emit(nuevoValor);
    }
  }
  
  reiniciar() {
    this.valor.set(this.valorInicial());
    this.cambioValor.emit(this.valorInicial());
  }
}
```
